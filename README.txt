###############

Project description: 
Terminal based program that provides the functionality of a social network. Program stores a network of user profiles that consist of information such as name, date of birth, zipcode, and a friends list. Upon running program, you can add users, add/remove friends, print all users, print the friends of a specific user, write the database's information to a file, find the shortest path between two users, and suggest friends to a user.

Included are sample index files of users to populate the network to test and show functionality.

###############

Compile instructions: 

user@computer~/path/to/files:$ make

###############

Usage instructions: user@computer~/path/to/files:$ ./social_network .txt
e.g., user@computer~/path/to/files:$ ./social_network users_large.txt
